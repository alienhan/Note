由于远程库是空的，我们第一次推送master分支时，

7.
将某个远程主机的更新，全部取回本地。
$ git fetch <远程主机名>
默认情况下，git fetch取回所有分支（branch）的更新。
如果只想取回特定分支的更新，可以指定分支名。
$ git fetch <远程主机名> <分支名>
比如，取回origin主机的master分支。
$ git fetch origin master
所取回的更新，在本地主机上要用"远程主机名/分支名"的形式读取。
比如origin主机的master，就要用origin/master读取。
