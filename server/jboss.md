安装




----------------------------------------------------------------------------
运行

1. bin/ add-user.bat
  添加用户

2. standalone.bat
  启动(单机模式)

3. 访问管理页面
  http://localhost:9990/
  登录:添加的用户

4. 服务端口
  http://localhost:8080/






----------------------------------------------------------------------------
关闭
   1启动 : bin/ jboss-cli.bat
   2连接 : cmd 中 connect localhost:9990/
   3. shutdown
